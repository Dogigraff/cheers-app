# Исправление белого фона и темной темы карты

## Что было исправлено:

1. **Темный фон страницы:**
   - Добавлен явный черный фон для `html` и `body` в `app/globals.css`
   - Контейнер карты теперь имеет `bg-black` вместо `bg-background`

2. **Темная тема для Яндекс.Карт:**
   - CSS фильтры применяются к `canvas` элементам
   - JavaScript код применяет фильтры после загрузки карты
   - Используется `invert(1) hue-rotate(180deg)` для инверсии цветов

3. **Автоматическое применение:**
   - Фильтры применяются каждую секунду (на случай динамического создания canvas)
   - Также применяются с задержками 100ms, 500ms, 1000ms после загрузки

## Как проверить:

1. Обновите страницу (Ctrl+F5 для полной перезагрузки)
2. Откройте DevTools (F12) → Console
3. Должно быть:
   - ✅ Черный фон страницы
   - ✅ Темная карта (инвертированные цвета)
   - ✅ Видны маркеры-эмодзи

## Если все еще белый фон:

1. **Очистите кеш браузера:** Ctrl+Shift+Delete
2. **Проверьте консоль:** Нет ли ошибок JavaScript
3. **Проверьте Network:** Загружаются ли CSS файлы

## Временное отключение фильтров:

Если карта все еще не работает, можно временно отключить фильтры:

В `components/map/map-view.tsx` закомментируйте `useEffect` с `applyDarkTheme`.
