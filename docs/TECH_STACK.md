# Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn/UI (Button, Card, Dialog, Drawer, Input, Toast)
- **Map:** `react-leaflet` (Leaflet CSS must be imported in global layout)
- **Icons:** `lucide-react`
- **Database:** Supabase (Client: `@supabase/ssr`)
- **State:** `zustand`
- **Animations:** `framer-motion` (for swipe cards)
