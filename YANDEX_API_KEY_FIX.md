# Исправление проблемы с API ключом Яндекс.Карт

## Проблема
Ошибка: `Invalid API key` - неверный или неактивный ключ Яндекс.Карт.

## Решение

### Вариант 1: Работать без ключа (для локальной разработки)

Я обновил код так, чтобы Яндекс.Карты работали без ключа для локальной разработки.

**Что сделано:**
- ✅ Код обновлен - API ключ теперь опциональный
- ✅ `.env.local` очищен от неверного ключа

**Просто перезапустите сервер** - карта должна работать без ключа.

### Вариант 2: Получить правильный ключ (для продакшена)

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите в аккаунт Яндекс
3. Создайте новый ключ JavaScript API и HTTP Геокодер
4. Добавьте в `.env.local`:
   ```
   NEXT_PUBLIC_YANDEX_MAPS_API_KEY=ваш_правильный_ключ
   ```
5. Перезапустите сервер
